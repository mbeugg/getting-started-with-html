HTML and The Web

What is HTML?

HTML is Hypertext Mark-Up Language. The vast majority of documents on the web are written in HTML.

What are the major pieces of the Web?

The Web connects the computers of individual users and their browsers to the internet, which is in turn connected to servers through http.

What are HTML tags?

HTML tags are commands that govern the look, style, and links of webpages written in HTML. Their syntax is usually of the form name contents /name, with greater than-less than signs around the name of the tag.

Some tags, especially those that insert links into webpages, have attributes as part of their command. These attributes can specify the address of the webpage, picture, or video. This video link is an example of an anchor tag with an attribute, and it does happen to be about html tags!.

What are the basic categories of HTML tags?

Most complete "elements" have both an opening and closing tag, but void tags have only the opening tag. Two examples of void tags are img, the image tag, and br, the break tag, which is kind of like a carriage return.

Another specialized category of tag is the block tag. The block tag appears to combine a command with a carriage return. This seems to follow from the fact that block tags create "invisible boxes" around the text or content they envelope.

The last (and perhaps largest?) category of tag is the inline tag. These tags apply their formatting to the document within the line in which they were inserted without adding any extra space. Some examples of these are a, span, br, img, and strong. This video link is a summary of inline vs. block elements.
